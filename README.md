## Getting Started

Welcome to the VS Code Java world. Here is a guideline to help you get started to write Java code in Visual Studio Code.

## Folder Structure

- `src`: sources
- `lib`: dependencies
- `bin`: compiled output

You can modify this structure via `.vscode/settings.json`.

## Dependency Management

Use the `JAVA PROJECTS` view in Visual Studio Code to manage dependencies.
